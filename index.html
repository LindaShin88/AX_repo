PROBLEM

기존 위원회 운영의 4가지 병목

대학 본부·단대·학과 단위 위원회 60여 개. 매 학기마다 반복되는 비효율은 운영 담당자·교원·외부위원 모두에게 누적됩니다.

📞
일정 조율

위원 7~15명에게 일일이 전화·메일·문자로 가능 시간을 받고, 엑셀에 받아 적어 최적 시간 선정

📅
시간표 확인

교원의 강의 시간을 학사 시스템에서 한 명씩 직접 조회. 트랙·학과 여러 곳 교차 검색 필요

✍️
외부위원 서명

회의록 인쇄 → 외부위원에게 우편/방문 요청 → 직접 서명 → 회수 → 스캔. 평균 3~5일 소요

📄
회의록 정리

참석 위원 명단·서명·일시·장소를 매 회의마다 수기로 정리하여 최종 PDF 보관

SOLUTION

4가지 핵심 가치

운영 담당자의 처리 동선, 일반 위원·외부위원의 응답 부담을 모두 줄이는 방향으로 흐름을 재설계했습니다.

⚡
운영자 1.5시간 처리

회의 정보 입력 → 시스템이 후보 일정·시간표 충돌·메일 발송·응답 집계·PDF 생성을 모두 자동화. 운영 담당자 개입은 시작 + 확정 두 번뿐.

🔗
일반 위원 로그인 불필요

메일·문자로 받은 1회용 응답 링크를 클릭하면 바로 응답 화면. 계정 등록·앱 설치·암호 입력 모두 없음.

📡
한성대 시간표 자동 인식

공개 시간표 페이지를 113개 전공 병렬 크롤링(약 1.9초)하여 교원의 강의와 겹치는 후보 시간은 처음부터 제외.

✍️
외부위원 재방문 제로

메일 링크에서 회의록 미리보기 + 손글씨 전자서명까지 한 번에. 서명 일자 자동 기록 → 최종 PDF에 자동 첨부.

USER FLOW

4단계로 끝나는 회의 운영

01
회의 생성

위원회 선택 → 회의 정보 입력. 후보 일정 자동 생성 + 시간표 충돌 회피.

02
위원 응답

전체 위원에게 응답 링크 자동 발송. 폰·PC 어디서나 1분 안에 응답.

03
일정 확정

전원 응답 + 단일 최고 → 자동 확정. 동률·정족수 미충족 → 운영자 선택.

04
서명·PDF

외부위원 전자서명 → 회의록 + 서명 페이지 자동 병합 → 최종 PDF.

DETAILS

개발하면서 풀어낸 디테일

한성대 디렉토리 연동
교원 이메일 자동 채우기

이름만 입력해도 한성대 통합검색에서 이메일·소속·사번 자동 입력. 동명이인은 전화번호로 자동 구분, 같은 사람의 여러 소속은 자동 병합.

스케줄링 알고리즘
날짜 × 시간 다양성 보장

후보 일정을 추천할 때 같은 날짜·같은 시간이 몰리지 않도록 그리디 방식으로 다양성을 우선 배치. 시간표 충돌은 점수가 아닌 사전 필터링으로 제거.

자동 확정 정책
전원 응답 → 즉시 확정

마지막 위원의 응답이 들어오는 순간 최고 가능 슬롯이 단일하면 자동으로 확정. 여러 슬롯 동률이면 운영자에게 선택 알림 메일 + 관리자 페이지 링크 발송.

전자서명 시스템
회의록 본문 + 서명 자동 병합

업로드된 PDF 회의록 뒤에 외부위원 서명 페이지를 자동으로 붙여 단일 PDF로 생성. 한글(.hwp) 업로드 시 안내 페이지 + 서명 페이지로 별도 결합.

알림 채널
메일 + SMS 복사용 텍스트

이메일은 SMTP로 자동 발송, SMS는 발송용 텍스트를 운영자가 복사해서 문자/카톡으로 전송. 일정 확정 시 SMS 텍스트 팝업 자동 노출.

위원 4-type 분류
교원·직원·학생·외부위원

위원 유형별로 입력 항목·시간표 자동 연결·서명 대상 여부가 다름. 교원만 시간표 크롤링, 외부위원만 전자서명 요청 대상.
