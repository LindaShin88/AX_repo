# 위원회 자동화 플랫폼

대학 내 60여 개 위원회 운영의 핵심 비효율(일정 조율, 외부위원 서명 수집, 회의록 작성)을 통합 자동화하는 SaaS 플랫폼.

> AX 프런티어 챌린지 ‑ 생성형 AI 활용 행정 혁신 사례 발굴 대회 출품작

## 핵심 가치

- **운영 담당자 1.5시간 처리** — 기존 4~6일 → 97% 단축
- **일반 위원은 로그인 불필요** — 메일 링크 1클릭으로 일정 응답 / 전자서명
- **시간표 자동 인식** — 한성대 학사 시스템 시간표 연동, 강의 시간과 충돌하는 후보 슬롯 자동 표시
- **외부위원 재방문 제거** — 링크 기반 전자서명, 서명 시각·IP 자동 기록

## 빠른 시작

```bash
npm install
npm run seed       # 데모 데이터 생성
npm start          # http://localhost:3000
```

운영자 로그인: `admin` / `admin1234`

## 4단계 흐름 (기획안 기준)

1. **회의 생성 (운영자)** — 위원회 선택 → 회의 정보 입력 → 자동: 후보일정 6개 생성, 시간표 충돌 자동 인식, 전체 위원에게 응답 링크 발송
2. **가능일정 응답 (위원)** — 메일 링크 클릭 → 후보별 가능/불가/조정가능 선택 → 제출
3. **일정 확정 + 서명 요청 (운영자)** — 응답 매트릭스 확인 → 시간 확정 → 자동: 전체 위원 안내 발송 + 서명 링크 발송
4. **서명 + PDF 생성** — 위원 서명 (캔버스) → 운영자가 회의록 작성 → PDF 자동 생성 (서명 이미지 + 메타데이터 포함)

## 기술 스택

| 영역 | 사용 기술 |
|------|----------|
| 런타임 | Node.js 24 LTS |
| 웹 프레임워크 | Express 5 |
| DB | SQLite (better-sqlite3) |
| 템플릿 | EJS |
| UI | TailwindCSS (CDN) + Pretendard |
| 서명 캔버스 | signature_pad |
| PDF | PDFKit (Malgun Gothic 한글) |
| 시간표 연동 | axios + iconv-lite (EUC-KR) |
| 인증 | express-session + bcrypt |

## 데이터베이스 스키마

`database.js` 참조. 핵심 테이블: `operators`, `committees`, `members`, `meetings`, `meeting_slots`, `slot_responses`, `member_tokens`, `signatures`, `notifications`.

## 한성대 시간표 연동

- URL 패턴: `https://info.hansung.ac.kr/fuz/sugang/gyo_h_siganpyo_nopasswd.jsp?as_sabun={사번}`
- 시간표 데이터는 fullCalendar.js 가 별도 엔드포인트(`gyo_sigan_main_data_nopasswd.jsp`)에서 로드
- **세션 인증이 필요**하므로 데모 환경에서는 mock 시간표를 사용 (`services/timetable.js`)
- 실제 운영시: 운영자가 본인 한성대 세션 쿠키를 한 번 등록 → 자동 연동 (확장 포인트 명시)

## 주요 파일

```
committee-platform/
├── server.js                    # Express 진입점
├── database.js                  # SQLite 스키마 + 연결
├── seed.js                      # 데모 데이터 생성
├── middleware/auth.js           # 운영자 인증 미들웨어
├── routes/
│   ├── admin.js                 # 운영자 라우트 (인증 필수)
│   └── public.js                # 위원 토큰 라우트 (인증 불필요)
├── services/
│   ├── timetable.js             # 한성대 시간표 크롤러 + mock
│   ├── scheduler.js             # 후보 일정 자동 추천 알고리즘
│   ├── notify.js                # 알림 발송 (현재 DB 로깅, SMTP/SMS 확장 가능)
│   └── pdf.js                   # 회의록 PDF 생성 (서명 이미지 포함)
└── views/                       # EJS 템플릿
```
